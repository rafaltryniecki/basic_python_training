This is basic introduction to python.
Packages you will need are numpy, pandas.
You can install them manually or by uv - easy python environment and package manager.
Guide on how to install uv:
https://docs.astral.sh/uv/getting-started/installation/
If you don't have python, uv will manage this for you. After installing uv, create virtual environment with the following command:

uv sync

And done!
